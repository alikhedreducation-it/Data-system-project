\# 🤖 AI-Enhanced Data Processing \& Decision Support System



\*\*MSIT 5910 Capstone Project – Version 1 (Prototype)\*\*



\---



\## 🚀 Project Overview

This system transforms raw data into actionable insights using:

\- Automated data cleaning

\- KPI dashboard

\- Interactive visualization

\- AI-based prediction



\---



\## 🎯 Key Features

\- Upload CSV datasets

\- Automatic data preprocessing

\- Business KPI dashboard

\- Region \& product filtering

\- Interactive charts (Plotly)

\- AI predictions using Linear Regression



\---



\## 🧱 System Architecture

The system follows a layered architecture:

\- \*\*Presentation Layer\*\* → Streamlit UI

\- \*\*Processing Layer\*\* → Data cleaning \& transformation

\- \*\*Analytics Layer\*\* → Visualization + AI model
